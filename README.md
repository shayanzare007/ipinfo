# Ipinfo

This gem help you give ip addr information.

## Install

Go to project folder and run this command :

```
gem install gem_file_name.gem # => change name to .gemfile name in project
```

## Example

Ip addr information :
```
require "ipinfo"

#include module
include Ipinfo

#Ipinfo::ip_info(ip => string)
Ipinfo::ip_info("8.8.8.8")

```
Your Ip information :

```
require "ipinfo"

#include module
include Ipinfo

#your ip info
Ipinfo::my_ip()

```
